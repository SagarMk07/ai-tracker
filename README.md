# AI Tracker

AI Tracker is a modern full-stack application designed to monitor, organize, and manage AI tools, workflows, and productivity systems in one place. It features a "Sacred Focus Room" to help users achieve deep work states.

Built with performance, scalability, and clean UX in mind.
